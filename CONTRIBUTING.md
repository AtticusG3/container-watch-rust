# Contributing

## Commits

Use [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` new feature
- `fix:` bug fix
- `chore:` tooling, deps, docs
- `refactor:` code change without behavior change

## Pull requests

- [ ] Builds on Windows: `cargo build --release --target x86_64-pc-windows-msvc`
- [ ] No secrets or local config committed
- [ ] ASCII-only operator strings in UI/logs
- [ ] `CHANGELOG.md` updated for user-visible changes
